Frequently asked

What makes Signal Market different from Google Alerts or RSS feeds? +
Google Alerts surfaces documents. Signal Market surfaces patterns. Every signal here has survived multi-source cross-validation — the confidence score tells you not just that something was mentioned, but whether independent data streams agree it matters. You don't get that from a feed.
Can I use Signal Market in my AI agent? +
Yes. The API v2 returns WorldStateObjects designed for agent consumption: typed fields, explicit nulls with documentation, confidence intervals, event classification. See the API docs for agent integration examples.
What data sources are monitored? +
Currently: arXiv (cs.AI, cs.LG, cs.CL), GitHub Trending, HuggingFace model trends, Alpha Vantage news sentiment, and arXiv RSS. Expanding to: Crypto signals, financial intelligence, and prediction market data in Q2 2026.
What does "cross-validated" mean? +
A signal is cross-validated when multiple independent sources confirm the same trend. An arXiv paper about AI Agents is one signal. When GitHub trending, HuggingFace model downloads, and arXiv all point to the same topic, the confidence rises and the signal is marked cross-validated. This is Signal Market's core value add.
What is the WorldState v2 API? +
WorldState v2 is an enhanced signal format designed for AI agent consumption. It adds: event classification, related domain mapping (from graph topology), causal phase estimation, and structured null fields that document what's coming next. It's an honest API — null fields mean "not yet computed" not "doesn't exist".
