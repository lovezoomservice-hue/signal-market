版本更新日志

v1.0.0-alpha 当前版本 2026-03-05
  • L0 数据接入层: 支持新闻API、社交媒体、政策文档等多数据源实时接入
  • L1 去噪层: 智能过滤垃圾信息、合并重复内容、质量评分
  • L2 事件图谱: 事件实体识别、因果关系推断、阶段检测
  • L3 概率引擎: 24小时/7天/30天概率曲线、驱动因子分析、预测市场集成
  • L4 API层: 完整的 RESTful 接口，包括事件、概率、简报、监控等
  • 用户透镜: A股板块(lens_a_stock)、美股宏观(lens_us_macro)、币圈事件(lens_crypto_event)
  • SDK支持: Python SDK、JavaScript/Node.js SDK
  • 性能优化: Pipeline 执行时间 ~1.8s，API 延迟 <2ms
