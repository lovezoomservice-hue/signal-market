Quick Start

Get started with Signal Market APIs in seconds:

# Fetch today's intelligence brief
curl https://signal-market-z14d.vercel.app/api/v2/agent-brief

# Get ranked signals with urgency filter
curl "https://signal-market-z14d.vercel.app/api/v2/rank?urgency=high&limit=5"

# Register for notifications (requires API key for auth endpoints)
curl -X POST https://signal-market-z14d.vercel.app/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "plan": "free"}'
Authentication: Most endpoints are public. Auth endpoints (/api/auth/*) require registration. Include your API key with X-API-Key header for authenticated requests.
